Hardhat Node with MetaMask setup

```bash
# Run local node
hh node

# Add custom network to MetaMask
# Network name : HHNetwork
# Default RPC URL : http://127.0.0.1:8545/
# Chain ID : 31337 or 1337
# Currency Symbol: ETH
# SAVE
# Select Network : HHNetwork
# Select an account -> Add account from hardware wallet -> Import Account -> Enter your private key
# Settings -> General -> Show native token as main balance

```
